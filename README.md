# Simulador de Consumo de Energia em Python

Projeto desenvolvido em Python para simular o consumo mensal de energia elétrica de aparelhos residenciais, calculando o gasto em kWh e o custo estimado na conta de luz.  

O objetivo é aplicar conceitos básicos de programação e introduzir fundamentos de engenharia elétrica de forma prática.

---

## Descrição

O programa solicita ao usuário o nome do aparelho, sua potência em watts, o tempo médio de uso diário e o valor do kWh.  
Com esses dados, ele calcula o consumo mensal e o custo estimado, exibindo um relatório no terminal.  

O sistema também permite cadastrar automaticamente aparelhos que não estejam previamente na lista.

# Funcionalidades

- Cadastro automático de aparelhos  
- Cálculo do consumo mensal em kWh  
- Cálculo do custo mensal em reais  
- Organização do código com funções  
- Relatório formatado no terminal  

# Fórmulas utilizadas

Consumo mensal (kWh) = (Potência × Horas por dia × 30) / 1000
Custo = consumo * Preço do kWh
